# Foodlink: Emergency Food Inventory Management

A hybrid desktop-to-cloud inventory system designed for high-stakes, real-time tracking of emergency food supplies.

## Key Features
* **JavaFX Interface:** A clean, responsive desktop GUI for rapid data entry and visualization.
* **AWS Integration:** Utilizes **AWS RDS** for centralized data and **AWS Lambda** for serverless backend logic.
* **Multi-Terminal Tracking:** Real-time synchronization across different terminals to ensure data integrity.

## Tech Stack
* **Language:** Java
* **GUI Framework:** JavaFX
* **Cloud Infrastructure:** AWS (RDS, Lambda, API Gateway)
